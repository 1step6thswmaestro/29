How to use?


▷ Camera Stream

: 연결된 카메라(안드로이드, 웹캠 등)를 통해 촬영된 사진들이 서버를 통해 표시. 어떤 사람과 닮았는지 확률 정보를 확인 가능.

▷ Registered Person

: 등록된 모든 사람들의 사진 및 이름 표기. x버튼으로 등록을 해제가능. 빨간색: Wanted / 파란색: Missing / 초록색: Normal

▷ Registration

: 이름, 종류, 이미지 파일로 찾을 사람 등록이 가능. 단, 파일은 오로지 이미지 파일로 제한됨.

▷ Find to File

: 사람 얼굴이 있는 파일(이미지 파일) 을 통해서, Unknown 혹은 등록된 사람인가 확인.

▷ Immediate Learning

: 웹캠을 통해서 직접 얼굴이 있는 사진을 여러장 찍어 서버에 전송. 즉석으로 등록 가능.
